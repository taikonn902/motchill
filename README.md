# 🎬 Motchill Clone

Một dự án web xem phim trực tuyến (Movie Streaming App) được xây dựng bằng **React + TailwindCSS + API phim**.  
Dự án lấy cảm hứng từ các nền tảng xem phim online, cung cấp giao diện hiện đại, mượt mà và trải nghiệm người dùng tốt.

---

## Demo
> https://filmchill.vercel.app/

---

## 🛠️ Công nghệ sử dụng

- ReactJS ⚛️
- TailwindCSS 🎨
- Axios 🌐
- Movie API (TMDB / TVMaze / custom API)
- React Router DOM 🚏
- React Icons 🎯

---

## Cài đặt & chạy dự án

Clone project:
```bash
git clone https://github.com/taikonn902/motchill.git
```

Dùng TMDB API, tạo file .env:

VITE_TMDB_API_KEY=your_api_key

## **Cấu trúc dự án**
src/
 ├── components/   # UI components
 ├── pages/        # Pages (Home, Detail, Search)
 ├── api/          # Call API
 ├── hooks/        # Custom hooks
 ├── assets/       # Images, icons
 └── App.jsx
